# Evolution of Web Design

An interactive web page showcasing the evolution of web design from 1996 to 2024 using era-accurate layouts, typography, styling trends, and UI patterns.

## Features
- Scroll-snapping timeline
- Era-specific UI styles
- Interactive timeline navigation
- Responsive layout
- CSS Grid, Flexbox, animations, gradients, glassmorphism, and more

## Tech Stack
- HTML
- CSS
- JavaScript
